Visual Studio Community 2017

Visual Studio Community 2017のインストール解説ページです。

ダウンロードからインストール、起動後のアカウント認証までを行います。

特にMikanライブラリでインストーラーを使う方は、必ず一度は起動してください。

ダウンロード

以下のサイトからダウンロードします。

https://www.visualstudio.com/downloads/

インストーラーの実行

ダウンロードしたインストーラーを実行します。

続行します。

しばらく待ちます。

インストールする環境を選ぶ

Mikanを使うにはC++によるゲーム開発を選択してください。それ以外の項目を任意で入れても構いません。

選んだらインストールボタンをクリックします。

インストールを見守ります。

インストール後再起動を求められたら再起動してください。

次のように再起動も求められなければアカウントを準備してください。

Microsoftアカウントの作成

以下ページを参考に、Microsoftアカウントを作成してください。

https://www.microsoft.com/ja-jp/msaccount/signup/default.aspx?Search=true

サインイン

サインインします。

Visual Studioを起動してMicrosoftアカウントのメールアドレスを入力します。

アカウントにログインします。

待ちます。

Visual Studioの起動

これで初回起動終了です。たまに同じようにサインインが発生します。

Mikanのインストール途中の場合は、一度終了してください。
