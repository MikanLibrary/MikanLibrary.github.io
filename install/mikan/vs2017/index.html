黒画面を出す

Visual Studio Community 2017を使って、Mikanライブラリを用いたプロジェクトを作ります。

その後、プロジェクトをビルドして、黒い画面を出します。

黒い画面が出ればMikanライブラリを用いて開発ができます。

Visual Studioの起動

Visual Studioを起動し、ファイルから新規作成を選択し、プロジェクトをクリックします。

プロジェクトの作成

左のテンプレートからウィザードを開くと、中央にMikanライブラリのウィザードが出現します。

ウィザードを選択し、プロジェクトの名前をつけてOKをクリックします。

プロジェクトが開けたら、右のソリューションエクスプローラーからソースを開き、プロジェクト名.cppをダブルクリックで開きます。

すると、すでにある程度のプログラムが書かれた状態になっています。

ここで上の緑三角のあるローカルWindowsデバッガーをクリックします。

ビルド

ビルドしてください。

黒いウィンドウ

すべてが正常にインストールできれいればビルドが成功し、以下のような真っ黒なウィンドウが出現します。

Mikanライブラリの導入終了

これにてMikanライブラリの導入は終了です。お疲れ様でした。
