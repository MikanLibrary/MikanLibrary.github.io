Mikanライブラリのインストール

Mikanライブラリのインストール解説ページです。

ダウンロードからインストールまでを行います。

ダウンロード

以下からダウンロードします。他のVisual Studioに対応するものは こちら からダウンロードしてください。

MikanInstaller2017_v2.1.5.zip

インストーラーの実行

ダウンロードしたZIPを解凍し、インストーラーを実行します。

DirectX SDKとVisualStudioがインストールされていれば、正常に実行できます。

DirectX SDKがインストールされているのにだめな場合は、再起動をしてみてください。環境変数 DXSDK_DIR が有効になっている必要があります。

VisualStudioがインストールされているのにためな場合は、一度起動してサインインしてください。ドキュメントフォルダ内に Visual Studio 2017 というフォルダが必要です。

インストール場所を選びます。通常はこのまま次に進んでください。

Mikanライブラリを用いたプロジェクトを簡単に作れるウィザードをインストールする場合、テンプレートを追加します。通常はこのまま次に進んでください。

インストールを見守ります。

インストールが一通り終わりました。

インストールが完了したので、そのまま終了してください。

確認

これでMikanライブラリを用いたゲーム開発の準備が整いました。

次にウィザードを用いて真っ黒な画面を出します。
