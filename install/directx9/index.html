DirectX 9 SDK

DirectX 9 SDKのインストール解説ページです。

ダウンロードからインストールまでを行います。

Mikan導入し始めた方は、まずVisual Studioをインストールしてください。

ダウンロード

以下のサイトからダウンロードします。

https://www.microsoft.com/en-us/download/details.aspx?id=6812

インストーラーの実行

ダウンロードしたインストーラーを実行します。

次へをクリックします。

利用規約に同意してください。

情報を送るかどうかです。とりあえずNoにしておきましょう。

インストール場所を選びます。このまま次に進みます。

インストールするコンポーネントを選びます。このまま次に進みます。

.NET Frameworkのインストール

このように.NET Frameworkが必要と言われたら、インストールします。

インストールを見守ります。

エラーが発生した場合

次のようなエラーが発生した場合、この1つ下を見てください。

エラーへの対処

コントロールパネルから、プログラムと機能を選択し、Microsoft Visual C++ 2010のRedistributableを2つともアンインストールします。

その後、もう一度インストーラーを実行してください。

インストール完了

無事インストールが完了しました。

再起動

Mikanのインストール中の場合、一度再起動しておいたほうが良いでしょう。

再起動後、Mikanのインストール途中の場合は、Mikanライブラリをインストールしてください。
